Mission

The mission of this Bootcamp is to take you from Zero to Hero in React. During this 3-month journey, you will learn everything in live classes and then implement it by building projects.

The curriculum is designed with the principle of `learn by building`, so there will be a lot of Assignments/Projects you will work on. So only register if you are ready for a grind.

After this bootcamp, you will be able to develop fast and scalable react frontend web applications from scratch.

FAQ

Are there any Prerequisites?

You will need to have some basic knwoledge of programming(if-else, loops, etc.) and some experience with HTML, CSS, and JavaScript, but you don't need to be an expert. We will do React from Day 1

Will you cover Node.js and Data Structures?

This is a 3-month intensive frontend React bootcamp, we will not be covering Node.js or Data Structures. This is a pure React in-depth hands-on Bootcamp.

Will there be any job guarantee after this Bootcamp?

NO, this is NOT a Job-guarantee Bootcamp. (No false promises). But we will make sure, that after this bootcamp you will be so good in React that interview questions will be a peice of cake for you. We will also conduct Mock Interviews to test your knowlege towards the end of bootcamp.

Can I join this Bootcamp only on phone without Laptop?

No, you will need a laptop as you will be working on assignments and coding projects.

Will I get a Certificate?

Yes, you will get a participation certificate after the bootcamp. Top performers will also get a LinkedIn recommendation from Akshay Saini.

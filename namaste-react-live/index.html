Frequently Asked Questions

What are prerequisites for the course?

- A laptop with Internet connection
- Basic Knowledge of HTML, CSS, JavaScript
- Hunger for Learning
- Ready for a grind

What language will the course be taught in?

Everything will be taught in English

Will there be recordings of the sessions?

You get 1 year access to the recordings of all the course videos

Will you cover Node.js and Data Structures?

This is a 2-month intensive frontend React Live course, we will not be covering Node.js or Data Structures. This is a pure React in-depth hands-on Live course.

Will there be any job guarantee after this Live course?

NO, this is NOT a Job-guarantee program. (No false promises). But we are sure, that if you finish these Live Classes and assignments on time, you will be so good in React that interview questions will be a piece of cake for you.

Can I join the classes only on phone without Laptop?

No, you will need a laptop as you will be working on assignments and coding projects.
